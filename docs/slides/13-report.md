# 13. 리포트 현황

**월간·분기별 A/S 리포트 생성 및 내보내기**

Phase 1 산출물 | 슬라이드 13 / 15

---

## 목적 & 핵심 가치

### 왜 만드나?

- 팀장·총괄이 **월간/분기별 A/S 건수, 비용, 처리율**을 한눈에 파악
- 본사 보상 금액 추적 리포트로 **정산 누락 방지**
- Excel/PDF 내보내기로 **본사 보고·내부 회의 자료** 즉시 생성

### 기존 vs 신규

| 기존 어드민 | 신규 통합 앱 |
|------------|-------------|
| 리포트 기능 없음 — 수동 Excel 취합 | 자동 리포트 생성 + 필터링 |
| 본사 보상 금액 별도 스프레드시트 관리 | 보상 추적 리포트 내장 |
| 형식 불일치, 담당자마다 다른 양식 | 표준 템플릿 Excel/PDF 내보내기 |
| 데이터 추출에 30분 이상 소요 | 원클릭 생성 (5초 이내) |

## UI 상세

### 필터 영역 [A]
- **기간 선택**: 월 단위 DatePicker (기본값: 당월) / 분기 선택 드롭다운
- **리포트 유형**: 월간 | 분기 | 본사 보상 — 드롭다운
- **담당자 필터**: 전체 | 개별 담당자 선택 (팀장: 본인 팀만, 총괄: 전체)
- **리포트 생성 버튼**: Primary, 클릭 시 서버에서 리포트 데이터 생성

### 리포트 목록 [B]
- 생성된 리포트 테이블 (최신순 정렬)
- 각 행에 **Excel 다운로드** / **PDF 내보내기** 아이콘 버튼
- 행 클릭 시 하단 미리보기 영역에 요약 표시

### 리포트 미리보기 [C]
- 선택된 리포트의 핵심 지표 카드 형태 표시
- A/S 건수, 완료율, 비용 합계, 본사 보상 청구액
- 담당자별 건수 분포 바 차트 (간이)

## 시나리오

### 시나리오 1: 월간 리포트 생성 및 Excel 다운로드
1. 팀장이 기간 `2026-04`, 유형 `월간` 선택
2. `[리포트 생성]` 클릭 → `POST /api/reports` 호출
3. 생성 완료 후 목록에 신규 행 추가
4. `[Excel ⬇]` 클릭 → `GET /api/reports/export?format=xlsx&id={id}` → 파일 다운로드

### 시나리오 2: 본사 보상 추적 리포트 PDF 내보내기
1. 총괄이 유형 `본사 보상` 선택, 기간 `2026-04`
2. `[리포트 생성]` → 보상 3축(우리 지출 / 고객 수납 / 본사 보상) 데이터 집계
3. 미리보기에서 보상 청구 총액 확인
4. `[PDF 📄]` 클릭 → `GET /api/reports/export?format=pdf&id={id}` → PDF 다운로드

### 시나리오 3: 담당자별 처리율 확인
1. 팀장이 담당자 `김민수` 선택, 기간 `2026-04`
2. 해당 담당자의 A/S 건수·완료율·평균처리일 리포트 조회
3. 미리보기에서 성과 지표 확인

## API 매핑

| 동작 | Method | Endpoint | 비고 |
|------|--------|----------|------|
| 리포트 목록 조회 | GET | `/api/reports?type={type}&period={period}` | type: monthly/quarterly/compensation |
| 리포트 생성 | POST | `/api/reports` | body: { type, period, assigneeId? } |
| 리포트 상세 조회 | GET | `/api/reports/{id}` | 미리보기 데이터 |
| Excel 내보내기 | GET | `/api/reports/export?format=xlsx&id={id}` | Content-Type: application/vnd.openxmlformats |
| PDF 내보내기 | GET | `/api/reports/export?format=pdf&id={id}` | Content-Type: application/pdf |
| 리포트 삭제 | DELETE | `/api/reports/{id}` | 총괄만 가능 |

## 확인 필요 사항

1. 본사 보상 리포트 양식 — 본사 제출 표준 포맷이 있는지?
2. 리포트 보관 기간 — 자동 삭제 정책 필요 여부
3. 분기 리포트 기준 — 회계 분기(1~3월) vs 자연 분기 확인
4. Excel 템플릿 — 기존 수동 작성 양식을 그대로 유지할지 신규 디자인?
5. 리포트 자동 생성 스케줄 — 매월 1일 자동 생성 기능 필요 여부
